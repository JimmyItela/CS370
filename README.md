Current &amp; Emerging trends in CS

Briefly explain the work that you did on this project: What code were you given? What code did you create yourself?

"This project focused on building and training a pirate intelligent agent using reinforcement learning and deep Q-learning within a maze environment. The goal of the agent was to learn how to navigate the maze and consistently reach the treasure while maximizing rewards. This project demonstrated how machine learning models can improve performance over time through experience rather than explicit instructions."

Connect your learning from throughout this course to the larger field of computer science:

"For this project, I was provided with starter code that included the maze environment, helper functions, and neural network structure. The provided code defined how the maze worked, how actions were executed, and how rewards were assigned. It also included classes such as TreasureMaze and GameExperience, which handled environment interactions and experience replay.
The main code I created was the implementation of the Q-training algorithm. I completed the training loop that:
Reset the environment each epoch
Selected actions using exploration and exploitation
Stored experiences in replay memory
Trained the neural network using batches of data
Tracked win rates and training performance
Updated the target network periodically
Through this process, I implemented the logic that allowed the intelligent agent to learn optimal pathfinding behavior."

What do computer scientists do and why does it matter?

"Computer scientists create solutions that improve efficiency, automate tasks, and solve complex problems across many industries. Their work matters because technology is integrated into nearly every aspect of modern life, including healthcare, transportation, finance, entertainment, and security.
In this project, the intelligent agent represents how computer science can be used to develop systems that learn and adapt over time. Instead of manually programming every possible solution, computer scientists design systems capable of learning from experience, which is increasingly important in modern AI applications."

How do I approach a problem as a computer scientist?

"This project reinforced a structured approach to problem solving:
Understand the problem and constraints.
Break the problem into smaller components.
Analyze the provided system and existing code.
Implement solutions incrementally.
Test and debug frequently.
Evaluate results and refine the solution.
During this project, I learned to focus on understanding how the environment and reward system worked before writing the training logic. By building and testing the algorithm step by step, I was able to identify errors and improve performance more effectively."

What are my ethical responsibilities to the end user and the organization?

"Computer scientists have ethical responsibilities to ensure that systems are reliable, fair, and safe for users. Intelligent systems can influence decision-making, so it is important to design algorithms that behave predictably and avoid unintended consequences.
In reinforcement learning specifically, ethical responsibilities include:
Ensuring models do not learn harmful or biased behaviors
Testing systems thoroughly before deployment
Being transparent about how decisions are made
Designing systems that prioritize user safety and organizational goals
This project highlighted how training data and reward structures directly influence system behavior. As a result, careful design and testing are essential to prevent unintended outcomes."
